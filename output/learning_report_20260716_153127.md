## 二、类似练习题

### 📝 题目

已知函数 g(x) = 2x² - 8x + 7，求 g(x) 的最小值及取得最小值时的 x 值。

### ✅ 答案

最小值为 -1，在 x = 2 处取得。
步骤：x = -b/(2a) = 8/(2×2) = 2，g(2) = 2×4 - 8×2 + 7 = 8 - 16 + 7 = -1

### 💡 解题思路

利用二次函数顶点坐标公式 x=-b/(2a) 求对称轴，代入原函数得最小值。注意系数 a=2, b=-8 代入时符号的处理。
