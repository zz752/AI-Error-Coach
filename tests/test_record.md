## 3. 测试步骤

### 3.1 步骤一：环境检查

```bash
# 确认 Python 版本
python3 --version

# 确认依赖已安装
python3 -c "import openai; print(openai.__version__)"

# 确认环境变量
echo $OPENAI_API_KEY | cut -c1-8"..."
```

### 3.2 步骤二：功能1 — 错题原因分析

对 `wrong_questions.json` 中的每道题执行分析：

```bash
python skill/scripts/analyze_error.py \
  --question "<题干>" \
  --student_answer "<学生答案>" \
  --correct_answer "<正确答案>" \
  --subject "<学科>"
```

### 3.3 步骤三：功能2 — 生成类似题

基于分析结果中的知识点，生成类似练习题：

```bash
# 直接传参
python skill/scripts/generate_question.py \
  --knowledge_points "<知识点列表>" \
  --subject "<学科>" \
  --error_type "<错误类型>"

# 从 JSON 文件读取（先保存 analyze_error 的输出）
python skill/scripts/generate_question.py --from-file analysis.json
```

### 3.4 步骤四：功能3 — 生成复习计划

基于分析结果中的错误类型和知识点：

```bash
# 直接传参
python skill/scripts/review_plan.py \
  --error_type "<错误类型>" \
  --knowledge_points "<知识点列表>" \
  --subject "<学科>"

# 从 JSON 文件读取
python skill/scripts/review_plan.py --from-file analysis.json
```

## 4. 测试用例详情与预期输出

### 用例 #1 — 数学二次函数（概念混淆）

**输入**：
```
学科：数学
题目：已知函数 f(x) = x² - 4x + 3，求 f(x) 的最小值。
学生答案：f(x) 的最小值是 3
正确答案：f(x) 的最小值是 -1，在 x = 2 处取得。
```

**预期输出（功能1 — 分析）**：
```json
{
  "error_type": "概念混淆",
  "error_reason": "学生混淆了函数值与顶点坐标的概念，将二次函数一般式中的常数项 c=3 直接当作最小值，未使用顶点坐标公式 x=-b/(2a) 进行正确计算",
  "knowledge_points": ["二次函数", "顶点坐标公式", "函数最值", "配方法"]
}
```

**预期输出（功能2 — 类似题）**：
```json
{
  "question": "已知函数 g(x) = 2x² - 8x + 7，求 g(x) 的最小值及取得最小值时的 x 值。",
  "answer": "最小值为 -1，在 x = 2 处取得。\n步骤：x = -b/(2a) = 8/(4) = 2，g(2) = 2×4 - 16 + 7 = -1",
  "explanation": "利用二次函数顶点坐标公式 x=-b/(2a) 求对称轴，代入得最小值"
}
```

**预期输出（功能3 — 复习计划）**：
```
第1天：回顾二次函数顶点公式 x=-b/(2a)，做 5 道基础求顶点坐标题
第2天：练习配方法将一般式化为顶点式，与公式法对比
第3天：学习二次函数图像性质：开口方向、对称轴、顶点、最值
第4天：二次函数最值应用题专项训练 3 道
第5天：二次函数与一元二次方程关系综合练习
第6天：原错题重做 + 2 道变式题
第7天：限时自测（5 道二次函数综合题）+ 错题归纳总结
复习知识点数：5 个 | 预估总时长：3.5 小时
```

### 6.2 注意事项

1. **API 响应时间**：单次分析约 3-8 秒，取决于 LLM 响应速度
2. **Token 成本**：每道题完整分析（三项功能）约消耗 2000-4000 tokens
3. **输出一致性**：LLM 的输出具有一定随机性，同题多次运行可能得到措辞不同但实质一致的结果（可通过降低 temperature 缓解）
4. **Prompt 调优空间**：若特定学科或错误类型的分析质量不佳，可直接修改 `skill/references/prompt_templates.md`，无需改动代码
